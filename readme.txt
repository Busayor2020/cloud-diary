A cloud-based digital diary that allows users to create and store personal journal entries securely within WordPress.

== Description ==

Cloud-Diary transforms your WordPress site into a personal journaling platform. Each user gets their own private digital diary space, securely stored in the WordPress database. Perfect for personal blogging, online courses, and membership sites.

= Key Features =

* Private cloud-based diary for each registered user
* Auto-saving functionality
* Simple and intuitive interface
* Mobile-responsive design
* Secure cloud storage

= Usage =

Simply add the shortcode `[cloud_diary]` to any page or post where you want the diary interface to appear.

= For Developers =

* Built following WordPress coding standards
* Extensible through WordPress filters and actions
* Clean, documented code
* Open source on GitHub

== Installation ==

1. Upload the `cloud-diary` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode `[user_notes]` in your pages or posts

== Frequently Asked Questions ==

= Can guests use this plugin? =

No, users must be logged in to create and save diary entries.

= Are the diary entries private? =

Yes, each user can only see and edit their own diary entries.

= Does it work with page builders? =

Yes, you can use the shortcode in any page builder that supports shortcodes.

= Is my data stored securely? =

Yes, all diary entries are stored securely in your WordPress database.
